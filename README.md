# pleymob.com

Pleymob landing page hosted on GitHub Pages.

## Setup

- **Hosting**: GitHub Pages (Jekyll)
- **Content**: Markdown files edited in Obsidian
- **Deployment**: Commits pushed via Claude Code

## Workflow

1. Edit `index.md` in Obsidian (`Docs/Projects/pleymob.com`)
2. Ask Claude Code to push changes
3. GitHub Pages automatically rebuilds the site
